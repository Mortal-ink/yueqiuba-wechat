# yueqiuba-wechat
一款基于微信小程序、集合实时篮球资讯的线上约球平台
“约球吧”--微信小程序用户端基于微信小程序原生框架设计实现约球系统的在线约球主要功能，通过集成虎扑篮球资讯API实现篮球资讯特色功能；商家端和后台管理端通过服务端提供的数据接口获取各自所需信息进行查询和管理功能；服务端采用Springboot框架进行开发，使用MySQL数据库对平台数据进行存储。
